uses_girder_wall_recipes
/datum/element/uses_girder_wall_recipes

An element for girders, wall barricades, etc. that makes them use wall construction recipes. Only really meant for recipes where you click on the girder with a stack of materials to make a wall.

Procs

attempt_recipeHas the user attempt the wall recipe. Assumes that the structure and stack are of valid types for the recipe.

check_girder_stateChecks if the girder state of the structure matches the required girder state of the wall recipe.
check_recipeChecks if the user can do the wall recipe.
get_main_recipeReturns the main wall recipe of the stack for the structure, if any.
get_material_recipeReturns the material wall recipe of the stack for the structure, if any.
