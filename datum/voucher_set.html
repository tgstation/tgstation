Voucher Set
/datum/voucher_set

A set consisting of a various equipment that can be then used as a reward for redeeming a voucher.

See Voucher Redeemer for where these sets are handled.

iconOptional, Icon of the set. Defaults to first item if not set.
icon_stateOptional, Icon state of the set. Defaults to first item if not set.
