Check to make sure our input vars are sensible

  1. Is our atom in which this reaction is occuring still intact?
  2. Do we still have reagents to react with
  3. Do we have the required catalysts? If you're adding more checks for reactions, this is the proc to edit otherwise, generally, don't call this directed except internally
