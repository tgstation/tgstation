proc
evaluate_borg
(/mob/living/silicon/robot/this_borg, /mob/living/silicon/robot/other_borg)

Checks if we should see a specific cyborg on our "network". Arguments are our borg, and another borg

Intended to allow borgs with the same AI to see eachother, syndicate borgs (scrambledcodes) with no AI to see eachother and not-syndicate borgs with no AI to see no one. Syndicate borgs connected to an AI will no longer see other syndicate borgs except ones also slaved to the same AI.
