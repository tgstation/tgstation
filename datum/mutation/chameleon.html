Proc Details

proc
on_attack_hand
(/mob/living/carbon/human/source, /atom/target, proximity, /list/modifiers)

Resets the alpha of the host if they click on something nearby.

Arguments:

  • source: The host mob that just clicked on something.
  • target: The thing the host mob clicked on.
  • proximity: Whether the host mob can physically reach the thing that they clicked on.
  • modifiers: The set of click modifiers associated with this attack chain call.

proc
on_move
(/atom/movable/source, /atom/old_loc, move_dir, forced, /list/atom/old_locs)

Resets the alpha of the host to the chameleon default if they move.

Arguments:

  • source: The source of the signal. Presumably the host mob.
  • old_loc: The location the host mob used to be in.
  • move_dir: The direction the host mob moved in.
  • forced: Whether the movement was caused by a forceMove or moveToNullspace.
  • [old_locs][/list/atom]: The locations the host mob used to be in.
